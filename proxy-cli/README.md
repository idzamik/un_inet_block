Перед началом работы желательно выполнить go mod tidy, а затем можно проверить доступные команды через:
go run . –help
go run . proxy –help 
go run . proxy xray –help 
go run . proxy xray conf –help

Для установки или обновления локального Xray надо использовать команду go run . proxy xray setup. Эта команда должна скачать релиз Xray для текущей платформы и подготовить локальный бинарник для дальнейшей работы.
Если скачать Xray из сети не получается, можно передать путь до локального файла в команду go run . proxy xray setup /полный/путь/до/файла.
В качестве локального источника можно использовать либо ZIP-архив релиза Xray, либо уже готовый бинарный файл xray.
После установки локальный бинарник должен лежать в директории ./bin/xray, а проверить его наличие удобнее всего через команду ./bin/xray version.
Если нужно удалить локально установленный Xray, используй команду go run . proxy xray remove.

Для работы со списком серверов используется команда proxy xray conf с одним из флагов -i, -u или -d, что хорошо укладывается в модель флагов и вложенных команд Cobra. Флаг -i означает установку списка, флаг -u означает обновление списка, а флаг -d означает удаление локального списка. Источник списка опционален.
Если источник не указан, используется встроенная ссылка по умолчанию.
Если источник начинается с http:// или https://, список загружается по сети.
Если источник передан как обычный путь, список читается из локального файла.
Чтобы установить список серверов из встроенного источника, используй команду go run . proxy xray conf -i.
